---
title: 1.6. Các phím tắt cần nhớ
sidebar_position: 7
---

Các phím tắt cần nhớ được trình bày dưới đây là các phím tắt mặc định sử dụng trong Google Colab.

<table>
<thead>
<tr class="ui-sortable">
<th class="wps-sort-row ui-sortable-handle">Mac</th>
<th class="wps-sort-row ui-sortable-handle">Window</th>
<th class="wps-sort-row ui-sortable-handle">Ý nghĩa</th>
<th class="wps-sort-row ui-sortable-handle">Lưu ý</th>
</tr>
</thead>
<tbody>
<tr>
<td>command+shift+P</td>
<td>Ctrl+Shift+P</td>
<td>Mở command palette, nơi chứa các lệnh bằng tên, phím tắt đi kèm, và cho phép tìm kiếm lệnh qua tên.</td>
<td>Phím tắt có ý nghĩa tương tự trong Jupyter Notebook, Visual Studio Code.</td>
</tr>
<tr>
<td>control+M+H</td>
<td>Ctrl+M+H</td>
<td>Mở bảng thiết lập phím tắt.</td>
<td>Có thể xem các lệnh và đổi phím tắt đi kèm ở đây.</td>
</tr>
<tr>
<td>esc</td>
<td>ESC</td>
<td>Chuyển sang command mode nếu đang ở editor mode.</td>
<td>command mode chỉ khác editor mode ở chỗ sử dụng phím tắt gọn hơn, ví dụ khi ở editor mode bạn nhấn "A" thì chữ A sẽ được gõ vào trong cell, còn ở command mode thì đây là thêm một cell mới ở phía trên, bạn đỡ phải nhấn thêm Control+M. Bài này chỉ nói phím tắt ở editor mode, trong quá trình học các bạn có thể tự tìm hiểu thêm phím tắt ở command mode.</td>
</tr>
<tr>
<td>control+return</td>
<td>Ctrl+Enter</td>
<td>Chạy cell hiện tại và giữ nguyên vị trí ở cell đó.</td>
<td>cell hiện tại, hay cell đang được chọn, thường là cell có viền đổ bóng.</td>
</tr>
<tr>
<td>shift+return</td>
<td>Shift+Enter</td>
<td>Chạy cell hiện tại và đi đến cell tiếp theo, nếu ở dưới không có cell nào thì tạo thêm một code cell mới.</td>
<td></td>
</tr>
<tr>
<td>option+return</td>
<td>Alt+Enter</td>
<td>Chạy cell hiện tại và tạo thêm một code cell mới ở ngay dưới.</td>
<td></td>
</tr>
<tr>
<td>control+M+A</td>
<td>Ctrl+M+A</td>
<td>Thêm một code cell mới ở trên cell hiện tại.</td>
<td></td>
</tr>
<tr>
<td>control+M+B</td>
<td>Ctrl+M+B</td>
<td>Thêm một code cell mới ở dưới cell hiện tại.</td>
<td></td>
</tr>
<tr>
<td>control+M+-</td>
<td>Ctrl+M+-</td>
<td>Phân tách cell tại vị trí trỏ chuột.</td>
<td>Control M -</td>
</tr>
<tr>
<td>control+M+M</td>
<td>Ctrl+M+M</td>
<td>Đổi code cell thành text cell.</td>
<td></td>
</tr>
<tr>
<td>control+M+Y</td>
<td>Ctrl+M+Y</td>
<td>Đổi text cell thành code cell.</td>
<td></td>
</tr>
<tr>
<td>control+M+D</td>
<td>Ctrl+M+D</td>
<td>Xoá cell hiện tại.</td>
<td></td>
</tr>
<tr>
<td>control+M+Z</td>
<td>Ctrl+M+Z</td>
<td>Hoàn tác thao tác trên cell.</td>
<td>Thao tác trên cell như xoá cell, đổi code cell thành text cell.</td>
</tr>
</tbody>
</table>
