title: 3.1. Giới thiệu về kiểu dữ liệu trong Python
sidebar_position: 1
---

Xem tài liệu chính thức trên trang document của Python tại đây: <a href="https://docs.python.org/3/library/datatypes.html" target="_blank" rel="noopener">https://docs.python.org/3/library/datatypes.html</a>

Bình thường ta có dữ liệu dạng chữ, dạng số, thì trong python cũng có các kiểu dữ liệu khác nhau như sau, chúng ta sẽ xem qua trước có những loại nào, và nói kỹ hơn vào bài sau:

<ul>
 	<li>Built-in data types:
<ul>
 	<li>Numeric Types</li>
 	<li>Iterator Types</li>
 	<li>Sequence Types</li>
 	<li>Text Sequence Type</li>
 	<li>Binary Sequence Types</li>
 	<li>Set Types</li>
 	<li>Mapping Types</li>
 	<li>Context Manager Types</li>
 	<li>Type Annotation Types</li>
 	<li>Other Built-in Types:
<ul>
 	<li>Functions</li>
 	<li>Boolean Values,...</li>
</ul>
</li>
</ul>
</li>
 	<li>Dates and times</li>
 	<li>Fixed-type arrays</li>
 	<li>Heap queues</li>
 	<li>Double-ended queues</li>
 	<li>Enumerations</li>
</ul>

Ở mức độ căn bản, ta chỉ tìm hiểu các kiểu dữ liệu built-in. Built-in mang ý nghĩa các kiểu dữ liệu này đã được cài đặt sẵn trong Python, chúng ta chỉ cần sử dụng lại, không cần thêm hay chỉnh sửa gì.

Để nói về kiểu dữ liệu, trước tiên ta cần có dữ liệu. Dữ liệu trong python sẽ được tự động gán kiểu built-in nếu nó được viết đúng. Để xem dữ liệu có kiểu built-in nào, ta dùng lệnh <code>type()</code>
